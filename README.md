# AlgoLounge

Interactive coding practice platform with structured courses and algorithm challenges.

## Quick Start

### First Time Setup

```bash
./setup
```

### Start the Application

```bash
./start
```

Open http://localhost:4200 in your browser.

## If Something Doesn't Work

**Frontend problems:**
```bash
cd frontend
npm install
npm start
```

**Backend problems:**
```bash
cd backend
npm install
npm run dev
```
